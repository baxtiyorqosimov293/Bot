# Telegram Photo Bot — single file version

Это упрощенная версия для загрузки с телефона.

## Что загружать в GitHub
- `main.py`
- `requirements.txt`
- `render.yaml`
- `ENV_EXAMPLE.txt`

## Переменные в Render
Скопируй значения из `ENV_EXAMPLE.txt` и добавь их в Render -> Environment.

## Запуск
Render:
- New -> Blueprint
- выбери репозиторий
- проверь что `render.yaml` найден
- добавь переменные окружения
- Deploy

## Что умеет бот
- 3 бесплатные генерации
- Telegram Stars
- 3 стиля
- 1 фото
- 2 фото в один кадр
- SQLite база на persistent disk
